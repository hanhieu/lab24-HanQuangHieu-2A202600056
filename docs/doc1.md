# Vietnamese Personal Data Protection Decree (Decree 13/2023/ND-CP)

## Overview
Decree 13/2023/ND-CP is the first comprehensive legal framework for personal data protection in Vietnam, effective from July 1, 2023. It establishes principles, rights, and obligations for personal data processing.

## Key Principles
1. **Lawfulness**: Personal data processing must have a legal basis
2. **Purpose Limitation**: Data must be processed for specified, explicit, and legitimate purposes
3. **Data Minimization**: Only collect data necessary for the stated purpose
4. **Accuracy**: Ensure personal data is accurate and up-to-date
5. **Storage Limitation**: Retain data only as long as necessary
6. **Security**: Implement appropriate security measures to protect data

## Personal Data Categories
- **Basic personal data**: Name, date of birth, nationality, marital status
- **Sensitive personal data**: Political views, religious beliefs, health status, biometric data, criminal records
- **General personal data**: Information about transactions, location, behavior

## Data Subject Rights
Data subjects have the right to:
- Be informed about data processing
- Access their personal data
- Request correction or deletion
- Object to processing
- Data portability
- File complaints with authorities

## Data Controller Obligations
Data controllers must:
- Register with the Ministry of Public Security
- Implement data protection measures
- Conduct impact assessments for high-risk processing
- Report data breaches within 72 hours
- Appoint a data protection officer for large-scale processing
