# Data Breach Notification Requirements

## Notification Timeline
Under Decree 13/2023/ND-CP, data controllers must report data breaches to the Ministry of Public Security within 72 hours of becoming aware of the breach.

## Required Information
The breach notification must include:
1. Description of the breach (nature, scope, cause)
2. Categories of personal data affected
3. Number of data subjects impacted
4. Consequences of the breach
5. Measures taken to address the breach
6. Contact information of the data controller

## Notification to Data Subjects
If the breach poses a high risk to data subjects' rights and freedoms, the data controller must notify affected individuals without undue delay. The notification should include:
- Description of the breach
- Likely consequences
- Measures taken by the controller
- Recommendations for data subjects to protect themselves

## Penalties for Non-Compliance
Failure to report data breaches can result in:
- Administrative fines up to 5% of total revenue
- Suspension of data processing activities
- Criminal liability in severe cases
- Compensation claims from affected individuals

## Best Practices
- Establish a breach detection and response team
- Conduct regular security audits
- Maintain incident response plans
- Train staff on breach reporting procedures
- Document all breach-related activities
